当前定位 这是 Agent 项目的独立展示页，用来展示自然语言请求如何经过 Planner、Tool 和最终回答链路。
当前已接通 社区最高星版本查询、官方推荐版本查询、版本时间线、版本内容、版本对比、治理缺口扫描。底层通过 gateway 进入 xiaogugit，推荐读路径优先走 Redis。

链路说明
Request 用户输入自然语言问题，并给出默认 project / filename 上下文。
Planner DMXAPI 负责选择最合适的 tool，并补齐参数，不直接查询底层数据。
Tool 本地 Python tool 调用 gateway，再由 gateway 转发到 xiaogugit / probability。
Answer LLM 基于 tool 结构化结果输出中文结论，避免直接编造 Git 事实。
